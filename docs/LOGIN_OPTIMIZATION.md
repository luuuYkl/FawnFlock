## 技术实现

### 组件结构
```vue
<template>
  登录卡片
    - App 头部（Logo + 名称 + 标语）
    - 选项卡切换器
    - 表单区域（登录表单 / 注册表单）
    - 提交按钮
    - 消息提示
    - 辅助操作
    - 用户协议
    - 第三方登录
</template>

<script>
  - 表单数据管理
  - 实时验证逻辑
  - 提交处理
  - 密码强度计算
  - 错误处理
</script>

<style>
  - 响应式布局
  - 动画效果
  - 主题色配置
</style>
```

### 状态管理
- `activeTab`: 当前激活的选项卡（login/register）
- `focusedInput`: 当前聚焦的输入框
- `showPassword`: 密码显示/隐藏
- `loading`: 提交加载状态
- `agreed`: 协议同意状态
- `loginForm`: 登录表单数据
- `registerForm`: 注册表单数据
- `errors`: 验证错误信息
- `message`: 提示消息

### API 集成
- 使用 `userAPI.login()` 进行登录
- 使用 `userAPI.register()` 进行注册
- 成功后保存 token、userId、username 到 localStorage
- 跳转到 HomePage
